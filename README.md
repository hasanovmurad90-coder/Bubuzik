Baby Taste (Food Tracker) - PWA
================================

1. Структура
------------
- index.html          — само приложение Baby Taste.
- manifest.webmanifest — манифест PWA.
- service-worker.js   — кэширование для оффлайн-режима.
- icons/              — сюда положите иконки:
    - icon-180.png    — 180x180 для iOS (apple-touch-icon).
    - icon-192.png    — 192x192 для Android/Chrome.
    - icon-512.png    — 512x512 для Android/Chrome.

2. Как подготовить иконки
-------------------------
- Нарисуйте квадратную иконку (например, в Figma/Canva):
  фон: #f8f9fa или белый,
  простой значок (например, эмодзи 👶 и/или 🥦),
  без мелкого текста.
- Экспортируйте PNG и сделайте три размера:
  180x180, 192x192, 512x512.
- Положите файлы в папку "icons" рядом с index.html.

3. Как запустить локально
-------------------------
PWA и service worker работают только через HTTP(S), а не через file://.

Вариант попроще (Python 3 установлен):
- Откройте папку "Baby Taste (Food Tracker)" в терминале.
- Запустите:
    python -m http.server 8000
- Откройте в браузере:
    http://localhost:8000/index.html

4. Как установить на iPhone
---------------------------
- Откройте страницу в Safari по адресу вашего сервера (например, http://192.168.x.x:8000).
- Нажмите кнопку "Поделиться".
- Выберите "На экран «Домой»".
- Подтвердите название "Baby Taste".

После этого приложение будет запускаться в полноэкранном режиме как обычное iOS-приложение и работать оффлайн (HTML/манифест/скрипты кэшируются, данные хранятся в localStorage).
